找零货币问题

在美国，就像在大多数国家一样，给任何总数的零钱最好的方法是使用一个贪婪的策略——找到面额最大但少于总数的硬币，给其中一个，然后重复。例如，在美国，支付给一个人 97 美分的现金，照此策略 会得出找零 50*1 + 25*1 + 5*2 + 1*2 的方法给出的硬币总数最少
然而，也有可能出现这种贪婪策略并不总是适用的硬币系统。例如，在一个奇怪的国家里，居民们出于某种奇怪的原因，决定使用 1、12、14、63 的面值。假设你需要返还 24 美分。最好的方法是返还 2 枚 12 美分的硬币。然而，在贪心策略下，总是选择面额小于总数的最大硬币，你会选择一个 14 美分的硬币和 10 个 1 美分的硬币，总共是 15 美分。

程序员必看纪录片

这个标题有歧义

想看纪录片了，好奇有哪些关于从 0 开始数数的世界的纪录片，搜了搜，发现有一篇「10部程序员必看的纪录片」的文章在第一页搜索结果中占了多数席，打算先从这 10 部里挑些有趣的看看，其余类似主题的也放在这里

现代生活的秘密规则：算法

里面说 Gale-Shapley 算法的一个实例是中国的高考志愿，我有空看看这是不是真的

操作系统革命

并没有提到很多具体的技术，全篇只是在很空洞的夸自由世界的东西，很有趣的地方是纪录片（好像是世纪初）中说中国是个第三世界的国家，就影像资料来看，确实很难否认，大家不理发，打地铺睡公司，用着大屁股显示器，看的都是大部头，封面上有只小企鹅
以及黑客原本的意思好像也是这部纪录片中获知的。

独立游戏大电影

好久以前白嫖过了，也许我该从 steam 上买一份？

代码奔腾

看的时候没有看明白，原来影片最后是在表明失败的现实，我是不在乎这些的，不是说视而不见当这种失败不存在，而是觉得有点理想主义很不错，况且，也并非完全的失败。

Patent Absurdity

毕达哥拉斯定理是个坏名字，勾股定理是个好名字；牛顿第一定律是个坏名字，经典力学第一定律是个好名字；爱因斯坦理论是个坏名字，相对论是个好名字；欧几里得算法是个坏名字，辗转相除法是个好名字；对于起好名字，程序员远比其他职业要在行，毕竟这是 ta们 的本职工作

给你命名的权力利已很不错，竟然还有脸要专利，真是不成体统。

我相信现在大多数人认为「通信的数学理论」是全人类共有的，而「一种通信的数学理论」就可以申请专利，这两者之间的区别在何处呢？我不知道，或许只有(1948-1971)年的差距罢了，从这个角度讲，对人类的贡献越大，专利期反而应该越短

科学家还真是好打发啊，给个 物理单位、数学名词、元素名 就完事了，怪不得小时候大家都想当科学家

开评论

挑选

在各个给静态网站提供评论功能的解决方案里面翻来翻去，有朋友建议我用 Disqus ，但是这个服务已经被墙了，我不可能只和程序员做朋友，所以选一个没有被墙的服务比较好，但用国内的服务又得来一遍 手机号实名备案 三连，干脆跳过不考虑。最后发现一个叫 gitalk 的，可以用 issue 当评论区用，就打算用这个了。

问题

这个帖子里说可能有危险，密钥直接明文我是知道的，其实也不要紧，公开仓库这种东西，本地肯定是有备份的

评论的认证机制比较麻烦，因为要自己开启评论区（还好，授权一次以后打开页面自动就会开启），我看项目的 readme 好像有个办法是给经常评论的人仓库的 contributor，这样 ta 们想说话的时候能自己初始化。不过不清楚是不是会受到上一段的影响

所有想评论的人都要从这里登录，然后允许给我设置的 oauth app 写所有公共仓库的权限，之后才能评论

有个问题是 issue 标题字符数有不能超过 50 的限制，光脚的不怕穿鞋的，我直接把 permalink 删成 :year/:mouth/:day 完事，反正一天不可能写两篇。

Postscript

有可能的话我还是希望能够耦合的少一些，不是弄个<script>生成在 html 里，而是每个页面引用同一份 js 文件，每个页面往 js 里送的参数只有自己的 issue 评论链接（我这里就是 permalink），其余的数据都配置在 js 里（能在生成时从 theme.xxx 取更好）
没学过这东西的逻辑，不明白该怎么写，以后再说

新标签页扩展

火狐自带的新标签页很不错，但是我想看 bing.com 的壁纸，理想中的新标签页和 bing 首页差不多，但

  • 壁纸还得有
  • 底部没有这么多推送，取而代之的是我的常访问网站，也就是自带新标签页的 Top Sites
  • 顶部没有 MS 其他产品的链接
  • 右上角无所谓，放点啥我都不注意，可能放今日天气或者别的自定义部件吧
  • 右边没有那个时不时出现，挡住屏幕 1/3 ，问我要不要参与推广活动的表格
  • 右下角那个简介默认展开
  • —— 可选 ——
  • 为了通用性，最好做成搜索引擎可以自定义
  • 为了通用性，最好做成壁纸 可以自定义（但这样就不知道壁纸简介放什么了）

然后我就开始找有没有符合我这些要求的扩展了，找了两天，没找到，有朋友建议我自己写一个，不知道他是不是说着玩，不过这确实是一种方法，所以我计划有时间的时候看一看各个新标签页相关的扩展

我不打算用油猴脚本，所以不会直接 魔改（魔这个字会不会来源于modify？） 必应

New Tab Override

可以把一个 网站/HTML 文件 设置为主页，但是很可惜我没找到符合要求的网站，所以没有用

Tabliss

看起来很好看，但是和我的想法相去甚远，没有用，竖着的屏幕用这个挺不错的

这个扩展用 React + TS 写的,源代码想生成成品扩展还要再下专门的包，看的我一脸懵逼

New Tab Splash

有点我设想中的东西的雏形，壁纸从 Splash 取，有菜单栏和搜索框。热门网站没看到从哪里设置，app 和 mail 两个菜单项似乎坏掉了

New Tab Tools

有最近关闭页面和热门网站，没有搜索框，壁纸只能手动从本地选

Infinity New Tab

能满足要求，但对我来说功能太多，太重（自带了很多图标），而且非常不“清真”：

  • 不是给火狐用的，介绍里面直接说了可能会出 bug

  • 介绍剩下的部分像是直接复制粘贴的，所有的 「Chrome」 都没有换成 「Firefox」

  • 不是 FOSS ，许可证那里写 所有权利保留

New Tab. Minimal.

感觉是实验性质的作品，但是正好有所有我需要的功能，许可证是 MPLv2 ，有可能的话打算以这个为基础改

下午两点过五分到两点半的树阴

这篇短文主要发表了对学校基建的负面意见，所以不只是植树计划的问题，不过每当我去上下午第一节课的时候，植树计划的问题确实是最大的问题

桂林的天气像个布尔值，如果东南沿海地区没有刮台风，天气就是特别毒辣的太阳，如果刮台风，天气就是以短暂的阴天为间隔的大雨，如果钻牛角尖，认为会有闪电劈到山上的雨和让人没有干袜子穿的雨是两种不同的天气，那就算三种。

假设你是个吸血鬼，从宿舍到校门口这么走一趟，就知道校园哪里该栽树了，很可惜，制定植树计划的生物里没有吸血鬼。

想到这里我气得浑身发抖，大热天的全身冷汗手脚冰凉，这个社会还能不能好了，我们吸血鬼到底要怎么活着你们才满意，眼泪不争气的流了下来，这个国到处充斥着对吸血鬼的压迫，吸血鬼何时才能真正的站起来。

艳阳天里就连走路去上课也像是某种试炼，非常令人不爽的是下课回来的时候还能看见工人们在别的地方种树。

这几天在下雨，所以我没什么要抱怨太阳的了，说点别的，图书馆的开水机，看着快坏了，三教不知道为什么总是特别热，风扇还偏偏开了也不转，等我以后发达了，一定给学校捐笔钱，专门指定用于图书馆和教学楼的基础设施维修工作
